## 📌 Project Overview

This project focuses on implementing the **complete Machine Learning Life Cycle (MLLC)** along with **Synthetic Data Generation (SDG)** and evaluating its effectiveness using the **TSTR (Train Synthetic Test Real)** methodology.

The goal was to:

* Build end-to-end ML pipelines
* Generate high-quality synthetic data
* Evaluate model performance using real vs synthetic datasets
* Ensure data privacy and scalability

* Generated realistic tabular datasets

### ✅ Benefits Achieved

* Data privacy preservation
* Reduced dependency on real datasets
* Improved model robustness

---

## 🧪 TSTR (Train Synthetic Test Real)

### Workflow:

1. Split real dataset → Train/Test
2. Generate synthetic data from training set
3. Train model on synthetic data
4. Test on real data
5. Compare with real-trained baseline
